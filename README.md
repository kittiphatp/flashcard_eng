# 📚 English Vocabulary Flashcard App

ยินดีต้อนรับสู่ Flashcard App เพื่อเรียนรู้คำศัพท์ภาษาอังกฤษผ่านภาพ!

## 🎯 วิธีใช้งาน

1. **เปิดไฟล์ index.html** ในเบราวเซอร์
2. **ดูภาพ** ของสัตว์ที่ปรากฏขึ้น
3. **พิมพ์คำศัพท์ภาษาอังกฤษ** ที่ถูกต้องในช่องInput
4. **กดปุ่ม "ตรวจสอบ"** หรือกด Enter เพื่อตรวจสอบคำตอบ
5. **ใช้ปุ่มนำทาง** เพื่อไปยังคำศัพท์ก่อนหรือถัดไป

## ⭐ ฟีเจอร์

✅ **ภาพตัวอักษร** - ภาพจริงของสัตว์แต่ละตัว
✅ **ระบบคะแนน** - ติดตามความคืบหน้าของคุณ
✅ **คำใบ้** - ข้อมูลเพิ่มเติมเพื่อช่วยจำ
✅ **แสดงความก้าวหน้า** - บาร์ความก้าวหน้าแบบเรียลไทม์
✅ **ตรวจสอบคำตอบ** - ฟีดแบ็กทันที
✅ **ทำใหม่** - เริ่มต้นเกมใหม่ได้ทุกเมื่อ
✅ **ตอบรับมือถือ** - ใช้ได้กับอุปกรณ์ทุกขนาด

## 🚀 การติดตั้ง

1. คัดลอกไฟล์ทั้งหมดไปยังโฟลเดอร์ที่ต้องการ
2. เปิด `index.html` ในเบราวเซอร์
3. เริ่มเรียนรู้!

## 🌐 เบราวเซอร์ที่รองรับ

- Chrome/Edge (ขอแนะนำ)
- Firefox
- Safari
- Opera

## 📝 หมายเหตุ

- ภาพได้มาจาก Unsplash API
- ที่ส่วนใหญ่ไม่ต้องมีการตั้งค่าเพิ่มเติม
- ตรวจสอบการเชื่อมต่อ internet เพื่อโหลดภาพ

## 🎓 เคล็ดลับการเรียนรู้

1. พูดคำศัพท์ออกมาดัง ๆ ขณะทำแบบฝึกหัด
2. ฝึกซ้ำ ๆ จนกว่าคุณจะตอบถูกทั้งหมด
3. ใช้คำใบ้เพื่อช่วยจำเพิ่มเติม
4. ตั้งเป้าหมายให้ตอบถูก 100%
